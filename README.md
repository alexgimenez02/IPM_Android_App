# Smart S-light Application
Smart S-light is the new project from Sofía Cárdenas, Blanca Cervera and Àlex Giménez
The project focuses on the problem of light pollution, changing the functionallity of street lights to consume less and reduce the amount of light pollution they produce.

First of all, the application focuses on the graph and animal-person interaction, with the use of media photos, people would be able to share animal sighting near their zone.
You can see the contacts you have inside this applications and the photos they have uploaded and where this photos were taken.

The application was build using Android Studio, the language is Java and the version is 'com.android.application' version '7.2.1', 'com.android.library' version '7.2.1'
